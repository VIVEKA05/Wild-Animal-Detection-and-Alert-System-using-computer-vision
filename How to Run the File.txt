# 📱 WhatsApp Alert Integration using Twilio

The AI Wildlife Detection System is integrated with the **Twilio WhatsApp API** to provide **instant real-time notifications** whenever a wild animal is detected. As soon as the AI model identifies an animal with high confidence, the system automatically sends a WhatsApp message to the registered mobile number containing the detected animal's name, confidence score, and the detection timestamp.

This feature allows farmers, forest officials, or security personnel to receive immediate alerts even when they are away from the monitoring system.

# Twilio Configuration

Before running the project, create a free Twilio account and configure the following credentials in the Python program.

```python
TWILIO_ACCOUNT_SID = "YOUR_ACCOUNT_SID"
TWILIO_AUTH_TOKEN = "YOUR_AUTH_TOKEN"

TWILIO_WHATSAPP_FROM = "whatsapp:+14155238886"
TWILIO_WHATSAPP_TO = "whatsapp:+91XXXXXXXXXX"
```

### Required Details

* Twilio Account SID
* Twilio Auth Token
* Twilio Sandbox WhatsApp Number
* Your Verified WhatsApp Number

> **Note:** During development, Twilio Sandbox requires your WhatsApp number to be joined to the sandbox before messages can be received.
